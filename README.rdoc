== TODO manager

Simple todo manager with tasks, task lists, priorities of tasks, statuses of tasks. 
Link to the {heroku}[http://todo-manager-100500.herokuapp.com/].

Were used:

* Rails

* EmberJS

* jQuery

* Handlebars

* Bootstrap

* Devise

* Active Model Serializers
